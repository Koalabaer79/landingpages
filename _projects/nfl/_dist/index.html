Inofficial NFL Website

by Lars Urban

Das ist eine inoffizielle Webseite zur Darstellung von Teams, Tabelle und Spielplänen / Spielbegegnungen. Das gesamte sichtbare Projekt wurde mit Javascript und JSON Datenen realisiert.

Zu dem gibt es ein kleines Backend, welches die Daten in den JSON Dateien pflegt. Dazu zählen die Teams, der Spielplan, die Punktestände.
